# zingfood-template
ZingFood P-rofessional Food Delivery Template

ZingFood is a modern, responsive, and visually appealing food delivery website template built using HTML5, CSS3, and Bootstrap 5. It is designed to provide a seamless user experience for browsing food categories, searching for dishes, and exploring top-rated meals.

Key Features
Professional Navigation: Includes a sticky navbar with a functional-style search bar and clear Login/Signup actions.

Search Bar Fix: High-visibility typed text ensured via optimized CSS for better usability.

Dynamic Hero Section: A high-impact hero banner with a gradient overlay to grab user attention immediately.

Enhanced Categories: Large, centered circular category images with smooth hover animations and scale effects.

Interactive Food Cards: Clean cards featuring diet indicators (Veg/Non-Veg), rating badges, and "Add to Cart" buttons with custom hover states.

Responsive Design: Fully optimized for mobile, tablet, and desktop screens using Bootstrap's grid system.

Professional Footer: Multi-column footer including social media icons, app store links, and legal navigation links.

Technologies Used
HTML5: Semantic structure for the layout.

CSS3: Custom styling, CSS variables for easy branding, and advanced hover effects.

Bootstrap 5: Responsive grid system and UI components.

Google Fonts: 'Poppins' for a modern and clean typography feel.

Font Awesome: Professional icons for search, branding, and social links.
